Requirements

In order to use the SDK you need to install Pi4j on your Raspberry.
Once you installed pi4j you can go further in this documentation.

Informations

Source code is available in the Github repo with examples. The javadoc will be generated and uploaded here : JavaDoc

Get the SDK

The SDK is available on jitpack, so add the jitpack repository in you maven repositories.

        <repository>
            <id>jitpack</id>
            <name>Jitpack</name>
            <url>https://jitpack.io</url>
        </repository>

Go to the jitpack page and check for the latest release or commit tag and add the dependency.

    <dependency>
        <groupId>com.github.coco33920</groupId>
        <artifactId>AdafruitSeesawJava</artifactId>
        <version>Tag</version>
    </dependency>

Basic Use

As a simple test and first step with the SDK you can recreate the BlinkExamples in the examples/gpio package. First, let's instantiate a Seesaw object :

    Seesaw seesaw = new Seesaw(I2CBus.BUS_1);

The first parameter of the constructor is the bus on which your device is connected, i use the first bus, but put in here the bus you use. You will need to init the device, call the method init() from the Seesaw object

    seesaw.init();

These firsts two lines are the same every times. After, you need an object to control the GPIOs of the Seesaw

    GPIOModule gpio = new seesaw.getGpioController();

you can also create a GPIOModule object with the seesaw :

    GPIOModule gpio = new GPIOModule(seesaw);

After, set the pin you want to use, like the pin 15 to OUTPUT like this :

    gpio.setMode(Modes.OUTPUT,Pins.GPIO_15);

The Pins class is an enum to wrap all of the differents context for the pins (GPIO, ADC, PWM, etc.) you could use just "15" but i recommend to use the enum. You can now make the LED blink with the setHigh(int) et setLow(int) methods of the GPIOModule :

    gpio.setHigh(GPIO_15);
    while (true) {
        Thread.sleep(1000);
        gpio.setLow(GPIO_15);
        Thread.sleep(1000);
        gpio.setHigh(GPIO_15);
    }

The speed of the blinking can be adjusted with the sleep calls. Deeper explainations are in the corresponding pages.
